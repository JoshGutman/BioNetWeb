About the Backend

BioNetFit is a command line centric program that takes input from users in the form of bngl (BioNetGenLanguage) and exp(Experiment) files. These files give specific parameters to the program, allowing it to simulate a real life molecular experiment. The program runs on the back of NFSim and BioNetGen, two systems that allow for easier access and processing of experimentation files.
