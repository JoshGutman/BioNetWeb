About the Backend

BioNetFit works by requiring the user provides a bngl and a exp (experiment) file. It is then scanned for the needed input values for the file to be ran. The BioNetFit user interface then allowes the user to easily navigate and change options and then allowing them to download the resulting file, or try to run the file directly from the server. These. It uses underlying programs to access the BioNetGen and NFSim programs, used to generate the results.
