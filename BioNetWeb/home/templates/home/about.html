About BioNetWeb

BioNetWeb is a Web 2.0 Grpahical User Interface for NAU's BioNetFit software. Created by Team U.I. Fit, the purpose of this website is to increase the usability and ease of access of BioNetFit via a user-friendly interface and graphically displayed results. In order to run the given experiments, the server makes use of a Monsoon vitrual enviroment, allowing for large experiments to be done
quickly and efficiently.

How it Works

1. The user uploads a either a singular bngl file, or a bngl file partnered with an exp file to the BioNetWeb server.

2. The user will be redirected to a parameters page. Here, they can fill in any information about the experiment that was not present within the given files.

3. Once the parameters are chosen, the experiment is run on the NAU Monsoon virtual enviroment. In some cases, this process may take hours.

4. When the experiment is complete, the server will return the results of the experiment, including graphs that will outline information such as best fit, and average.
